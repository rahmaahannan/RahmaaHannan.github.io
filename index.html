How to set colours in HTML.

Problem: I think a pink header would totally look awesome. I don't know how, though.

Answer: For a pink header you add "style="background-color:" inside your header bracket. Followed by the hex code of the colour you want.

Example: "< h1 style="background-color:#FFBFF8;">PINK>/h1>"
