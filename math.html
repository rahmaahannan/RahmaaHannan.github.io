VECTORS (8/11/2025)

This is from Edexcel Linear 1MAO Vectors. I attempted the whole worksheet successfully.The difficulty level was 2/5. Solely, because it was difficult writing down my directions since my writing kept getting mixed up with the question itself. Additionally, some prerequisites are the basics of vectors. Just Unit Vector Form. The question that interested me and I am focusing on is Q6

i) The first question asks us to find OS in terms of p and q. To do this I first found PQ, which was from OP backwards to OQ. (-p + q). The question then tells us "S is the midpoint of PQ." Using this information I now know PS and SQ are both (1/2)(q-p). This made it easier to find OS since I can now go from OP to PS without any unknown values. This means that OS = OP + PS which is p + (1/2)q + (1/2)p. Which leads us to our answer that OS = (1/2)q + (1/2)p.

ii) The second question tells us to show RS is parallel to OQ. To do this, I know that RP = (1/2)p, this is useful since to get to RS we need to go from RP to PS. We found out previously that PS = (1/2)(q-p) This means RS = (1/2)p - (1/2)p + (1/2)q. If we simplify it we get RS = (1/2)q. Luckily, OQ is given to us (OQ = q) This means that 2RS = OQ. We have now successfully shown RS is parallel to OQ.
